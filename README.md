IoT Weather Station using ESP32

Overview

This project monitors environmental conditions using sensors and sends data through an ESP32 microcontroller.

Features

- Temperature Monitoring
- Humidity Monitoring
- Real-time Sensor Data
- IoT Connectivity
- Low Power Operation

Hardware Components

- ESP32 Development Board
- DHT22 Temperature and Humidity Sensor
- Breadboard
- Jumper Wires

Software Requirements

- Arduino IDE
- ESP32 Board Package
- DHT Sensor Library

Working Principle

The DHT22 sensor measures temperature and humidity. The ESP32 reads the sensor values and displays them through the Serial Monitor. The data can also be uploaded to cloud platforms for remote monitoring.

Applications

- Smart Homes
- Agriculture Monitoring
- Weather Forecasting
- Environmental Monitoring

Conclusion

The project demonstrates the use of IoT and embedded systems for real-time weather monitoring.
